## 🖥 Projecto

Esse é um projecto web responsivo de uma cápsula do tempo para exibir memórias em uma linha do tempo.

## 🚀 Tecnologias

Esse projecto foi desenvolvido durante o NLW da Rocketseat com as seguintes tecnologias:

- HTML
  -CSS
  -Git e Github
